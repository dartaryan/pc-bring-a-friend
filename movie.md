## 12. Tips for Success with Veo 3

### Best Practices

**1. Iterative Generation**
- Generate 2-3 variations of each segment
- Select the best elements from each
- Don't expect perfection on first try

**2. Prompt Engineering**
- Start with detailed prompts
- If results are off, simplify
- Add specific details one at a time
- Use reference images if Veo 3 supports them

**3. Consistency Across Segments**
- Use similar prompt structure for all 4
- Mention "continuing from previous segment" in later prompts
- Keep lighting and style keywords consistent

**4. Common Issues & Fixes**

**Issue:** Generated video is too generic
**Fix:** Add more specific brand details, colors, and style references

**Issue:** Text in video is unreadable
**Fix:** Generate without text, add in post-production

**Issue:** Pacing is wrong
**Fix:** Specify exact timing for each action in prompt

**Issue:** Colors don't match brand
**Fix:** Include exact hex codes and emphasize "strictly adhere to brand colors"
